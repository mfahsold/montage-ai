# Security Policy

We take security seriously. Please report vulnerabilities responsibly.

## Supported Versions

We currently maintain `main` as the active development branch. No formal LTS branches exist yet.

## Reporting a Vulnerability

- Email: [security@montage-ai.dev](mailto:security@montage-ai.dev) (or open a private advisory via GitHub Security Advisories if available).
- Please include:
  - A description of the issue and potential impact
  - Steps to reproduce or a proof-of-concept
  - Affected commit/versions and environment details
- We aim to acknowledge reports within 72 hours.

## Disclosure

We prefer coordinated disclosure. After a fix is available, we will credit reporters who wish to be acknowledged.
