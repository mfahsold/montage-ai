# Montage AI Documentation

> **We polish pixels, we don't generate them.**

Free, open-source, privacy-first AI rough-cut tool for creators and professionals.

## Quick Start

```bash
# Local development
make dev && make dev-test

# Kubernetes cluster
make cluster
```
