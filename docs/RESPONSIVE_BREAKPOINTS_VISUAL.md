## CSS Classes by Breakpoint

### Mobile (Default)
```css
.grid-cols-1              /* Single column */
.grid-cols-auto           /* Minmax(300px, 1fr) */
.flex-col                 /* Vertical */
.gap-4                    /* 1rem */
.px-1                     /* 1rem padding */
.mb-8                     /* 2rem margin */
```

### Tablet (md: 768px+)
```css
.md:grid-cols-2           /* Two columns */
.md:grid-cols-3           /* Three columns */
.md:flex-row              /* Horizontal */
.md:gap-6                 /* 1.5rem gap */
.md:px-2                  /* 2rem padding */
.md:items-center          /* Center alignment */
.md:justify-between       /* Space between */
```

### Desktop (lg: 1024px+)
```css
.lg:grid-cols-3           /* Three columns */
.lg:grid-cols-4           /* Four columns */
.lg:flex-row              /* Row layout */
.lg:gap-8                 /* 2rem gap */
.lg:px-2.5                /* 2.5rem padding */
.lg:max-w-1400            /* Max width */
```

```css

.xl:gap-8                 /* 2rem gap */
```
