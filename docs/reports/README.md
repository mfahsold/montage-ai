# Historical Reports Index

This index groups one-off session notes, readiness summaries, and dated validation reports.

Use this folder as a navigation layer. The report files remain at their current paths for compatibility.

## Canonical vs Historical

- Canonical docs: `docs/README.md`, `docs/getting-started.md`, `docs/configuration.md`, `docs/roadmap/BACKLOG.md`
- Historical reports: point-in-time snapshots that may be outdated after later releases
